## Getting Started

```bash
npm install
npx convex dev   # prompts for Convex login/project setup, creates .env.local
npm run dev      # in a separate terminal
```

First run of `npx convex dev` will open a browser to create/login to your Convex account and set up the project.

Open [localhost:3000](http://localhost:3000).
